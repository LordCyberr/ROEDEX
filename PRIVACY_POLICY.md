# ROEDEX Privacy Policy

**Effective Date:** June 17, 2026

ROEDEX ("we," "our," or "the extension") is committed to protecting your privacy. This Privacy Policy explains how we handle data when you use the ROEDEX Chrome Extension.

## 1. Zero Data Collection
ROEDEX collects **absolutely zero** personal data. 
- We do not ask for your email, passwords, or game credentials.
- We do not use tracking scripts, telemetry, or analytics software.
- We do not transmit any of your information to external servers or databases.

## 2. Local Storage Only
All data generated by the extension—including but not limited to your loot history, session statistics, Gold/XP rates, tracked items, and UI settings—is stored **100% locally** on your device using your browser's local storage capabilities (`localStorage` and `IndexedDB`). 

If you uninstall the extension or clear your browser data, your ROEDEX data is permanently deleted. We have no way to access, recover, or view your data.

### 2.1 Debug Export Feature
ROEDEX includes an optional "Export Safe Debug Report" feature in the developer panel. This feature exists strictly for you to troubleshoot performance issues. If you choose to export this report, the resulting JSON file contains **no personal information**. It strictly records:
- **UI Settings**: Layout configurations, scales, and active tabs.
- **Performance Diagnostics**: Current FPS, estimated RAM usage, WebSocket packet counts, and React rendering speeds.
- **Metric Counts**: The *number* of entities loaded in memory (e.g., `mobsTracked: 15`, `playersTracked: 2`), not the identities or details of those entities.
This JSON file is generated locally and downloaded directly to your hard drive. It is **never** automatically sent to us or any external server.

## 3. Passive Operation
ROEDEX operates passively. It intercepts and reads the WebSocket traffic already being sent to your browser by the game client in order to calculate statistics. It does not inject commands, automate gameplay, or send automated network requests back to the game server.

## 4. Permissions
To function, ROEDEX requires specific permissions in your browser:
- **Host Permissions (`*://embervault.ruyui.com/*`)**: Required strictly to run on the Roots of Embervault website and monitor the game's WebSocket traffic. We do not request or have access to any other websites you visit.
- **Storage**: Required to save your settings and statistics locally on your machine.
- **Scripting**: Required to overlay the UI onto the game window.

## 5. Third-Party Links
ROEDEX does not share data with any third parties, as we do not possess any data to share. 

## 6. Changes to This Policy
Because ROEDEX is open-source, any future changes to how data is handled will be visible in the public GitHub repository and documented in our changelog.
