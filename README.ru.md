## 🌟 Обзор

**ROEDEX** — это премиальное, ненавязчивое расширение-оверлей для игры *Roots of Embervault*, работающее на стороне клиента. Пассивно считывая входящий WebSocket-трафик, ROEDEX предоставляет игрокам мгновенный доступ к игровой статистике в реальном времени, таймерам спавна, журналам добычи и интерактивным компаньонам.

Созданный с использованием **Manifest V3**, **React** и **Tailwind CSS**, инструмент предлагает великолепный плавающий интерфейс в стиле Glassmorphic, соответствующий по качеству дашбордам из AAA-игр.

---

## ✨ Ключевые возможности

| Функция | Описание |
| :--- | :--- |
| **📈 Отслеживание сессии** | Расчет опыта в час (XP/hr), рунных камней в час и общей ценности золота/добычи в реальном времени. |
| **🤖 Интерактивные компаньоны** | 4 уникальных компаньона (**Bob**, **Kaya**, **Lia** и **Crash**), динамически реагирующих на игровые события. |
| **🗺️ Высокоточные спавны** | Отслеживание расстояния и точной очереди респавна ресурсов, NPC и боссов на основе официальных данных карт. |
| **⚔️ Прочность снаряжения** | Мониторинг прочности оружия и брони в реальном времени с предупреждениями о критическом износе. |
| **🌐 Мультиязычность** | Полный перевод на **английский, испанский, русский и корейский** языки с локализованными гайдами. |
| **🎨 Темы Glassmorphism** | Стильные темы Obsidian Gold, Hologram и Ruby Glass с возможностью перетаскивания и открепления окон. |
| **📐 Динамическая настройка** | Полное изменение размера окон в 8 направлениях, смена разметки (вертикальная/горизонтальная) и сохранение позиций. |
| **⚡ Полная оптимизация** | Отсутствие утечек памяти (проверено на сессиях 12+ часов), сжатые медиа-ресурсы и легковесный билд Vite. |

## 📢 Что нового в v0.0.4

*   🌍 **Полная локализация:** Все оставшиеся интерфейсы (доска квестов, кузнец, список игроков) переведены на все поддерживаемые языки.
*   🐛 **Исправление бага с HUD сундука:** Устранена проблема, из-за которой закрытие сундука во время перемещения предметов приводило к зависанию HUD минимальной ценности на экране.
*   ⌨️ **Сохранение горячих клавиш:** Кастомные бинды (блокировка UI, смена разметки, сброс размера) теперь корректно сохраняются при перезапуске расширения.
*   ⏱️ **Задержка при запуске:** Оверлей теперь ожидает первого активного игрового пакета (с красивым кинематографичным появлением в течение 5 секунд) перед отображением.

## 🎮 Горячие клавиши

Следующие горячие клавиши могут быть полностью настроены во вкладке **Настройки (Settings)**:

| Действие | Стандартный бинд | Описание |
| :--- | :--- | :--- |
| **Свернуть / Развернуть HUD** | `Ctrl + Shift + M` | Превратить весь интерфейс ROEDEX в небольшую плавающую сферу. |
| **Сменить режим разметки** | `Shift + H` | Переключение между вертикальным (колонки) и горизонтальным (строки) отображением. |
| **Сбросить размер оверлея** | `Shift + R` | Сбросить размеры всех окон до стандартных оптимизированных настроек. |
| **Заблокировать / Разблокировать UI** | `Shift + U` | Зафиксировать окна на месте и включить режим клика «насквозь» для удобства в бою. |

## 🔒 Безопасность и конфиденциальность

Мы верим в абсолютную прозрачность. **ROEDEX собирает РОВНО НОЛЬ пользовательских данных.**

*   **100% Локально:** Все настройки, история добычи и кастомная разметка сохраняются только на вашем компьютере с помощью IndexedDB/localStorage.
*   **Без инъекций кода:** ROEDEX не внедряет скрипты в игру и не изменяет игровой клиент. Полностью безопасно для античитов.
*   **Пассивное чтение:** Расширение выступает исключительно зрителем входящего WebSocket-трафика игры для вывода информации.
*   **Ограниченные разрешения:** Запрашивается доступ только к официальному домену игры.

Подробности можно найти в нашей [Политике конфиденциальности](PRIVACY_POLICY.md).

## 🛠️ Инструкция по установке

### Вариант А: Web Store (Рекомендуется)
1. Откройте страницу **ROEDEX** в Chrome Web Store.
2. Нажмите **Установить (Add to Chrome)**.
3. Закрепите расширение на панели инструментов вашего браузера.
4. Запустите игру; расширение инициализируется автоматически!

### Вариант Б: Режим разработчика (Сборка из исходников)
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/LordCyberr/ROEDEX.git
   ```
2. Перейдите в папку проекта и установите зависимости:
   ```bash
   npm install
   ```
3. Запустите компилятор:
   ```bash
   npm run build
   ```
4. Откройте Chrome и перейдите по адресу `chrome://extensions/`.
5. Включите **Режим разработчика** (переключатель в верхнем правом углу).
6. Нажмите **Загрузить распакованное расширение (Load Unpacked)** и выберите созданную папку `dist`.
