# ROEDEX Features

## Tracking
- Real-time WebSocket parsing and event interception
- Entity Tracking (Mobs, NPCs, Nodes)
- Resource Tracking & Respawn Timers
- Session XP and Gold tracking

## UI
- Premium Glass UI with Backdrop Blur
- Vertical & Horizontal Layout Modes
- Customizable Widget Density (Compact vs Normal)

## Themes
- Built-in Theme System (Default, Ocean, Crimson, Neon)
- Dynamic neon glow matching selected themes

## Overlays
- Weapon Durability Overlay
- Armor Durability Overlay
- Floating Notification Toaster
- Floating Minimized Mode Orb (with connection status glow)

## Notifications
- Event-based notification system
- Drag-to-position custom layout mode
- Animated entry/exit toast system

## Persistence
- LocalStorage based Session Cache
- Draggable UI position persistence
- Settings persistence across reloads
